Snailshock

A 2-player competitive aim-and-shoot game, where you have to wipe out all the other player’s snails to win!

This project was made for a school project with a friend of mine, where I mostly worked on the controls and UI. This included mapping controller inputs to the UI as well as the player movement, ensuring that both gamepad and keyboard controls are supported for smooth multiplayer gameplay.

Engine: Unity
Language: C#
Role: Multiplayer controls and UI
Grade: A
